TaskMaster is a quest-style productivity app for Android. It allows users to create tasks, organize them by category, priority, and difficulty, track completed work, use a focus timer, view a daily calendar, and check player-style progress stats.

Features:
- Add, edit, complete, and delete tasks
- Categories, priority, and difficulty levels
- Due date selection
- Search, filter, and sort tasks
- Player Profile / progress stats
- Completed task history
- Calendar / daily planner view
- Focus timer with focus and break mode
- Light and dark mode support

How to Run:
1. Open the project in Android Studio.
2. Let Gradle sync finish.
3. Select an emulator or Android device.
4. Press Run.

Notes:
This project was created for a Mobile Programming final project. The app uses a quest/game-inspired theme to make task management more motivating.
